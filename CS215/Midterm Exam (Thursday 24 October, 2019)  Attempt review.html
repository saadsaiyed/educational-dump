Question text

What is static, relative, and absolute positioning in CSS?

Static: This position is given to all objects by default. It makes sure that the object is at the position as it is written in the html document (following the appearance at the code).


Relative: This enables the ability to use top, bottom, etc. and also it has the border associated by its parent object. Other than that it is same as static.


Absolute: This position takes the object and forces out from the html and put it over any object by overlapping on it. Also it will be removed from it original place making room for next object to take the empty place.
